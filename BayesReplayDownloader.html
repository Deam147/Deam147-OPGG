Bayes Replays

Una caracteristica del API de Bayes Esports es poder obtener las replays (.rofl) de las partidas

Esto aporta a poder hacer Vod Reviews desde el cliente de espectador, por lo tanto he desarrollado scripts para guardar automaticamente las replays en Drive (Organizadas por liga, fecha y jornada)

También desarrolle una aplicación para descargar las replays directamente al ordenador, con filtros de liga, fecha, equipos y demás
